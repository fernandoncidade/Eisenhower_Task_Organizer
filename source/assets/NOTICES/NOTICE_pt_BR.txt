Aviso de Licenças e Atribuições de Software de Terceiros

Este documento lista as principais bibliotecas de terceiros utilizadas no projeto EISENHOWER ORGANIZER e as referências às suas licenças. O objetivo é cumprir requisitos de distribuição (incluindo conformidade LGPL para bibliotecas Qt/PySide6) e oferecer informações para usuários e integradores. As dependências do projeto estão listadas no arquivo requirements.txt incluído no repositório/instalador.

Informações gerais sobre PySide6 (Qt para Python) — conformidade LGPL
- Biblioteca: PySide6 (Qt for Python)
- Licença: GNU Lesser General Public License (LGPL) v3 (ou conforme versão do PySide6 empacotada)
- Versão utilizada (ver requirements.txt): PySide6 6.x (conforme pacote no instalador)
- Substituição das bibliotecas LGPL:
  - As bibliotecas PySide6 são distribuídas como bibliotecas dinâmicas (DLLs) no instalador/executável.
  - Para substituir as DLLs PySide6, localize a pasta de instalação/instância que contém os binários PySide6 e substitua os arquivos DLL correspondentes por suas versões compatíveis.
  - Se desejar o código-fonte correspondente à versão do PySide6 usada no executável, ele está disponível publicamente em:
    https://github.com/qt/pyside
  - Caso haja dificuldade em obter código-fonte ou binários compatíveis, solicite instruções pelos canais de suporte informados no instalador/distribuição.

Declaração sobre transmissão de dados e privacidade
- O executável padrão não envia telemetria automática a servidores externos.
- Os dados do usuário (por exemplo, tasks.json e arquivos exportados) permanecem sob controle do usuário e são armazenados localmente.
- Para detalhes sobre tratamento de dados pessoais, consulte a Política de Privacidade incluída no instalador/distribuição.
